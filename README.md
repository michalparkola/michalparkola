# A few things from me

🧵 [Tapestry Skills](https://github.com/michalparkola/tapestry-skills) – a set of Agent SKILLS, my most popular contribution on Github
- 🏆 a skill to [turn a youtube video or blog post into a concrete action plan](https://github.com/michalparkola/tapestry-skills?tab=readme-ov-file#0-learn-this-)
- 🪓 a skill to [turn vague stuck tasks into concrete next actions](https://github.com/michalparkola/tapestry-skills?tab=readme-ov-file#5-unblock-action) in under 2 minutes
- 📒 a skill to [log a session with your agent](https://github.com/michalparkola/tapestry-skills?tab=readme-ov-file#4-session-log) saving highlights and optionally intermediate "chunks" of work that can be developed later

🔬 [apocomito](https://github.com/michalparkola/apocomito) – a Windsurf workflow (can be easily adapted for other agents) to define and test the hypotheses that need to be true for a project to succeed, based on a very useful model by Jason Cohen (included)
